EpiSuite

EpiSuite is a minimalist web framework. I built it for use with Socialbib and it helped us complete the site within 4 weeks. I've authored countless web sites, applications and web services over the years. They've all had the same thing in common...frameworks (or lack thereof) which almost made sense. EpiSuite represents the simplest way to retrieve a resource and send it to the user. Resources are what people are searching for and can come in the form of web pages, search results or a PDF file. EpiSuite aims to convert the traditional page per request mentality with a resource per request one.
