EpiSuite

EpiSuite started out as a minimalist web framework (EpiCode). I built it for use with Textbook Revolt and it helped us complete the site in 4 weeks. EpiSuite has since been used for several other projects and has grown to include the following.

  • EpiCode - An exceptionally fast and easy to understand MVC framework.
  • EpiDatabase - A database wrapper with catchable exception handling.
  • EpiForm - A client and server side form validator which supports both Prototype.js and YUI.
  • EpiCurl - An asynchronous HTTP library which uses curl.
