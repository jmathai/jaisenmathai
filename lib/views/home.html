PubliciTweet

The latest project I am working on is a Twitter marketing tool named PubliciTweet. As part of the project I wrote a library for the Twitter OAuth API in order to better understand the OAuth specification. That library is available on Github as is the documentation. One of the biggest challenges of this project was to efficiently make tens of thousands of API calls while honoring Twitter's API limits.
