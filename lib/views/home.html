PubliciTweet

A recent site I created was Twitter marketing tool named PubliciTweet. As part of the project I wrote a library for the Twitter OAuth API in order to better understand the OAuth specification. That library is available on Github as is the documentation. One of the biggest challenges of this project was to efficiently make tens of thousands of API calls while honoring Twitter's API limits.

I ended up selling the site in order to focus on the student to student textbook rental platform mentioned above. Before selling the site it had already sent 700,000 direct messages on Twitter and averaged over 40% click throughs.
